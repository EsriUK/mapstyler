Mapstyler help - colour

This app styles your map using a colour palette from COLOURlovers. Click random to get a new colour palette to re-style the map.

Clicking the individual colours in the bottom right allows you to change your colour palette and then update the map's style.

When you're happy, click the heart button in the top right to save a copy of this map to your ArcGIS Online account for use in your own maps and apps.

Built using Esri's dark gray canvas vector tile layer, version 4 of the ArcGIS API for JavaScript and RainbowVis-JS.
