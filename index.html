An app to quickly style vector tiles.

Developed by Esri UK, mapstyler allows users to style Esri vector tiles using images. Users can tweak colours before hitting the save button, creating a new basemap in ArcGIS Online.

Built using Esri's dark gray canvas vector tile layer, version 4 of the ArcGIS API for JavaScript, Color Thief, RainbowVis-JS and Spectrum.
